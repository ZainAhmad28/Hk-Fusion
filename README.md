# HK_Fusion

**HK_Fusion** is a modern and stylish ecommerce website designed for jackets, shoes, and glasses. This repository contains the complete HTML, CSS, and JavaScript code for the HK_Fusion online store. The website features a clean and responsive design, making it accessible on desktops, tablets, and mobile devices.

## Features

- **Product Display:** Showcases jackets, shoes, and glasses with images, descriptions, and prices.
- **Add to Cart:** Users can add products to the shopping cart.
- **Payment Options:** Supports mock payment methods like Cash on Delivery (COD) and online payment placeholders.
- **Responsive Layout:** Fully responsive design for all devices.
- **Clean Design:** Modern and minimal interface for easy navigation.
- **Policies & Terms:** Includes pages for Return Policy, Privacy Policy, and Terms & Conditions.
